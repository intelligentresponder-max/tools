# CONTENT-ROUTINE — Inhalte erstellen und posten (ueberall)

## Formate die Claude erstellen kann
- Social Media Post (Instagram, LinkedIn, TikTok, X)
- Story-Text (Bildunterschrift, Swipe-Karte)
- Reels-Script (Hook, Body, CTA)
- Newsletter / E-Mail
- WhatsApp-Broadcast
- Blog-Artikel
- Offline-Flyer / Aushang (PDF)
- Praesentation (PPTX)
- Audio-Script (Podcast, Hypnose, Erklaervideo)
- Druckvorlage (fuer Visitenkarte, Plakat, Aufkleber)

## Der universelle Content-Ablauf

### 1. Intent klaeren (Claude fragt automatisch)
- Wo soll es hin? (Plattform / offline / beides)
- Wer liest/sieht es? (Zielgruppe)
- Was soll passieren? (liken / kaufen / anrufen / einfach geniessen)
- Ton? (locker / professionell / hypnotisch / provokant)

### 2. Claude erstellt
- Hook: erste Zeile muss sitzen — stoppen, nicht scrollen
- Body: Kern-Botschaft, knapp, auf den Punkt
- CTA: was jetzt? (Link, Antwort, Speichern, Teilen)

### 3. Plattform-Regeln auf einen Blick

| Plattform | Laenge | Besonderheit |
|---|---|---|
| Instagram Post | 150-300 Zeichen sichtbar | Hook in Zeile 1, Hashtags am Ende |
| Instagram Story | 1-2 kurze Saetze | Grossschrift, Emoji okay |
| TikTok Caption | max 150 Zeichen | Hook + 3-5 Hashtags |
| LinkedIn | 1300 Zeichen optimal | Professionell, persoenliche Story |
| X (Twitter) | 280 Zeichen | Meinung, kein Weichspueler |
| WhatsApp Broadcast | kurz + direkt | Kein Spam-Ton, persoenlich wirken |
| Newsletter | 300-600 Woerter | Subject-Line = Hook, 1 CTA |
| Flyer/Offline | max 50 Woerter sichtbar | Headline + Tel/QR, fertig |

### 4. Output-Formen
- Nur Text → Claude gibt den fertigen Text
- Text + Bild → magix.py fuer Glow-Effekt (MAGIX-ROUTINE.md)
- Text + Video → magic_zoom.py (MAGIC-ZOOM-ROUTINE.md)
- PDF/Flyer → PPTX oder HTML als Druckvorlage
- Mehrsprachig → Claude uebersetzt direkt mit

### 5. Offline / just for fun
Kein Posting geplant? Trotzdem wertvoll:
- Als PDF speichern → drucken
- Als MP3-Script → selbst einsprechen
- Als Ideen-Sammlung → spaeter weiterverwenden
- Einfach Spass haben — kein Zwang zur Veroeffentlichung

## Goldene Content-Regeln
1. Hook first — die ersten 3 Worte entscheiden
2. Ein Post = eine Botschaft (nicht fuenf auf einmal)
3. Persoenlich schlaegt perfekt — echte Geschichten wirken
4. Zahlen und Fakten machen Texte glaubwuerdig
5. CTA immer — auch wenn es nur "Speichern lohnt sich" ist

## Merksatz
Hook. Body. CTA. Plattform kennen. Ton treffen. Fertig.
